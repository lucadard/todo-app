TODO APP from FrontendMentor: https://www.frontendmentor.io/challenges/todo-app-Su1_KokOW
React-app made with Vite.

Dependencies:
    *React,
    *ChakraUI,
    *react-beautiful-dnd

Main features:
    *responsiveness
    *add new todos
    *mark todos as complete
    *delete todos
    *filter todos
    *clear completed todos
    *light/dark mode
    *drag and drop to reorder items
    *save page state
